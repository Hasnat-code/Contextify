# 📄 Contextify

> Your AI-powered document companion. Upload any document and get precise, context-aware answers instantly.

![Contextify Banner](https://contextify-two.vercel.app/og-image.png)

## ✨ Features

### 🔍 RAG-Based Intelligent Search
Upload PDFs, Word docs, and more. Get instant, context-aware answers across your documents using advanced Retrieval-Augmented Generation.

### 💬 Ask Questions in Plain Language
No complex syntax needed. Ask your documents questions in natural language and receive accurate, relevant answers immediately.

### 📚 Multi-Format Support
- 📄 PDFs
- 📝 Word Documents (.docx)
- 📋 Text Files
- 📑 And more

### 🚀 Fast & Accurate
Powered by cutting-edge AI technology for instant processing and reliable results.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn
- Git

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/Hasnat-code/Contextify.git
cd Contextify
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
# or
pnpm install
```

3. **Set up environment variables**
Create a `.env.local` file in the root directory:
```env
NEXT_PUBLIC_API_URL=your_api_url
# Add other required environment variables
```

4. **Run the development server**
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

5. **Open your browser**
Navigate to [http://localhost:3000](http://localhost:3000) to see the application.

## 📖 How to Use

1. **Sign Up / Sign In**
   - Create an account or sign in with existing credentials
   - Secure authentication ensures your documents are private

2. **Upload Documents**
   - Click "Upload your first document"
   - Select PDFs, Word docs, or other supported formats
   - Wait for processing to complete

3. **Start Chatting**
   - Ask questions about your documents in natural language
   - Get instant, context-aware answers
   - Create multiple chats for different documents

4. **Manage Documents**
   - View all uploaded documents in the Documents panel
   - Organize and manage your document library
   - Create new chats for different use cases
